Learn Laser Interferometry

(A self-study course on interferometry for precision measurements, using IPython notebooks.)

Daniel Töyrä
Daniel Brown
Andreas Freise
and others
since 2016

This page provides resources and self-study material on laser interferometry. In particular we cover the topics related to the use of optical systems for gravitational wave detectors such as LIGO. If you are post-graduate student, an advanced undergradaute students or just interested in this topic, the following material provides a hands-on activities from basic interferometers such as optical cavities or to complex systems such as gravitational wave detectors.
We are in the process of uploading material. Expect frequent updates during March and April 2016

Get started!

The course is presented as a selection of IPython notebooks (or Jupyter notebooks), interactive Python notebooks that run in a web browser. You can interact with the notebooks in two ways:

  • Download the GitHib repository: The complete course is available in the GitHub repository Learn Laser Interferometry on GitHub. Follow the instructions in GitHub to download a local copy of the repository. In the main folder further installation instructions are provided in the file Introduction.pdf. This requires the installation of the certain Python packages but allows you to edit the notebooks and do the hands-on tasks.
  • View the notebooks on GitHub: If you want to browse the material before doing the tasks, or look up examples later on, you can use the direct links below to find the individual notebooks. The notebooks have been organised to present an easily browsable reference, start with the README file.

About the course

At the Gravitational Wave Group in Birmingham, UK, we work on the optical design and technologies of the laser interferometers for gravitational wave detection. For example, we led the optical design of the Einstein Telescope, the Optical Simulation and Design group towards the first baseline design of Advanced Virgo and are contributing to the Advanced Interferometer Configurations group of the LIGO Scientific Collaboration. In this course we combine material we have used in the past to train our graduate students starting in this field, for example as part of the GraWIToN training network.
